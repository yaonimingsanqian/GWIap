 GWIap 功能
=========================
简化苹果app内购买

使用方式
------------------------
见demo

安装
------------------------
### 源码安装

1. 直接下载文件，将源文件集成到你的工程

### cocopods安装
1. 未安装cocopods请自行安装.
2. 在terminal中执行pod search GWIap.
3. 编辑Podfile,添加 pod 'GWIap','~> 0.0.1'
4. 执行pod install --verbose --no-repo-update,后面加了参数，目的是快速安装.
